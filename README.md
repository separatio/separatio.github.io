# separatio.github.io

Personal site for Alex Rădulescu — AI-First Builder & Operator.

Built with Vite + React + TypeScript. Self-hosted IBM Plex Mono and IBM Plex Sans (woff2). Deployed to GitHub Pages via the `gh-pages` branch.

Sections: Hero, About, Experience, Building, Writing (with `/writing` route for articles), and Contact.

## Development

```sh
pnpm install
pnpm dev
```

## Build

```sh
pnpm build
```

## Deploy

Push to `main`; the GitHub Pages workflow publishes from `dist/` via `gh-pages`.
